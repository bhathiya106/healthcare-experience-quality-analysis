# Healthcare Experience Quality Analysis

A data science project analysing how patient comfort factors 
influence healthcare experience quality using Python and Machine Learning.

## Tools Used
Python, Jupyter Notebook, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SciPy

## Key Highlights
- Spearman correlation & Kruskal-Wallis statistical testing
- Logistic Regression vs Random Forest classification
- Patient satisfaction prediction
